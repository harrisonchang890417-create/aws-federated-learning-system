# AWS Federated Learning System

This repository contains multiple AWS cloud computing projects focused on distributed systems, federated learning, edge computing, and autoscaling applications.

## Technologies Used

- AWS EC2
- AWS Lambda
- AWS S3
- AWS SQS
- DynamoDB
- AWS IoT Greengrass
- MQTT
- Python
- Distributed Machine Learning

---

## Repository Structure

### federated-learning
Distributed federated learning workflow using EC2 workers, Lambda aggregation, and S3-based model synchronization.

### edge-computing
Edge computing implementation using AWS IoT Greengrass and MQTT-triggered federated learning workers.

### elastic-face-recognition
Elastic cloud-based face recognition system with autoscaling web and application tiers using SQS, DynamoDB, and EC2.

---

## Key Features

- Distributed model training across multiple workers
- Event-driven cloud and edge computing pipelines
- Autoscaling application architecture
- Federated learning on AWS infrastructure
- IoT Greengrass edge deployment
